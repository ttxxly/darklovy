根据书籍 ID 获取该书籍的正版源。

- 接口地址：/btoc
- 返回格式：json
- 请求方式：GET
- 请求示例：http://api.zhuishushenqi.com/btoc?view=summary&book=548d9c17eb0337ee6df738f5
- 请求参数说明：
  - id: 具体小说的ID
- 返回参数说明：该书籍的简介
- Json 返回实例

```
[
    {
        "_id": "568fef99adb27bfb4b3a58dc",
```
