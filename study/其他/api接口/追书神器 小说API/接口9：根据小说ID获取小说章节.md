- 接口地址：/mix-atoc/:bookId
- 返回格式：json
- 请求方式：GET
- 请求示例：http://api.zhuishushenqi.com/mix-atoc/50bff3ec209793513100001c?view=chapters
- 请求参数说明：
  - id: 具体小说的ID
