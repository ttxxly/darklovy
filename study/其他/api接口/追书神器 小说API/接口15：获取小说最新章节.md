模糊搜索

- 接口地址：/book
- 返回格式：json
- 请求方式：GET
- 请求示例： http://api05iye5.zhuishushenqi.com/book?view=updated&id=531169b3173bfacb4904ca67
- 请求参数说明：`view`: updated `id`:以都好分割的bookId
- 返回参数说明：
- Json 返回示例：

        "_id": "531169b3173bfacb4904ca67",
