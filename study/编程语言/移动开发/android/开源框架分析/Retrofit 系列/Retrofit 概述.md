Retrofit 是 美国移动支付公司 Square 公司发布在 Android 平台上 HTTP 访问 的开源项目框架。

项目地址：https://github.com/square/retrofit/ 

它是一个 RESTful 的 HTTP 网络请求框架的封装。Retrofit 2.0 开始内置 OkHttp，Retrofit 专注于接口的封装， OkHttp 专注于网络请求的高效安全，两者分工协作。
