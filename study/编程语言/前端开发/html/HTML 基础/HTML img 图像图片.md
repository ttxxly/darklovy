img 元素向网页中嵌入一幅图像。

```
使用格式
	<img 属性="值" 属性="值" ... />
注意：
	<img>标签并不会在网页中插入图像，而是从网页上链接图像。
	<img> 标签创建的是被引用图像的占位空间。
```

#### img 标签必须属性

```
alt：
	格式：alt="text"
	描述：规定图片的替代文本
src:
	格式：src="/i/eg_tulip.jpg"
	描述：规定显示图像的 url
```

#### img 可选属性

```
height：
	格式：height="value"
	两个可选值：
		% 
		pixels
	描述：定义图像的高度
width：
	格式：width=value"
	两个可选值：
		% 
		pixels
	描述：设置图像的宽度

align: 不推荐使用
	格式：align="top"
	有五个可选值：
		top：上
		bottom：下
		middle：中
		left：左
		right：右
	描述：规定如何根据周围的文本来排列图像
border: 不推荐使用
	格式：border="pixels"
	描述：定义图像边周围的边框距
hspace：不推荐使用
	格式：hspace="pixels"
	描述：定义图像左侧和右侧的空白
```

**注意：`img` 标签支持 HTML 全局属性和事件属性**
