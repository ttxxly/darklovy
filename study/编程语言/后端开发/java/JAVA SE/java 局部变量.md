关于局部变量你需要了解的

```
1. 在内存的栈中。
2. 随方法的调用而存在，随着方法的调用完毕而结束
3. 没有默认值，必须复制才能使用。
```

#### 相关面试题

```
1. 下列代码中的错误时（D）
    public class Test {
        public static void main(String [] args) {
            int i;
            i+=1;
        }
    }
    A. 非法的表达式 i+=1
    B. 找不到符号i
    C. 类不应为public
    D. 尚未初始化变量i
    解析：局部变量没有初始化过程，在使用前必须赋值。
```
