异常：程序执行时发生的问题

异常发生的原因有很多，通常包含以下几大类：

* 用户输入了非法数据
* 要打开的文件不存在
* 网络通信时连接中断，或者JVM内存溢出

我们可以使用 `try` 和 `catch` 关键词来捕获异常，将 `try/catch` 代码块放在异常可能发生的地方。

`try/catch` 代码块的一般格式：

```
try{
    //代码块
}catch(异常类型，例如：Exception e){
    //需要抛出的异常，例如：e.printStackTrace();
}catch(异常类型){
    //需要抛出的异常
}finally{
    //必定执行的代码块
}
```
